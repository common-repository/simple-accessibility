Make your website accessibility with just a few clicks.

== Description ==

Mit nur wenigen Klicks kannst Du deine Webseite barrierefrei machen.
Dieses Plugin nutzt Cookies, um die Einstellungen des Plugins zu speichern.

= English =

Make your website accessibility with just a few clicks.
This plugin use cookies to save settings of the plugin.

== Installation ==

1. Lade `simple-accessibility` in den Wordpress Plugin Ordner `/wp-content/plugins/`
2. Aktiviere `simple-accessibility` im Plugin Menü von Wordpress
3. Lese Dir unter "Einstellungen -> Simple Accessibility" die Anleitung durch

= English =

1. Upload `simple-accessibility` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read Construction under "Settings -> Simple Accessibility"
